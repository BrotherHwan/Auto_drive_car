# Auto_drive_car
## 프로젝트 목적
초음파 센서와 DC motor를 통해 장애물을 피해 자율운행하는 자동차를 만듭니다.<br/> 
## 사용 디바이스
stm32f411, 초음파 센서, DC moter, 블루투스 모듈, 안드로이드 휴대폰
## 개발환경
STM32CubeIDE, arduino bluetooth application(휴대폰에 설치)
